# **App Name**: Paisano Sales Hub

## Core Features:

- User Authentication and Roles: Secure user authentication using Firebase Authentication with role-based access control (Admin, Seller, Manager).
- Lead and Client Management: Comprehensive CRM functionality to manage prospects, clients, and sales opportunities with custom fields for 'Número de cliente', 'Nombre del cliente', 'Región', etc.
- Sales Pipeline Kanban Board: Interactive Kanban board to visualize and manage the sales pipeline, allowing drag-and-drop functionality to move deals through different stages.
- Quotation Management: Manage and version quotations with PDF upload and storage via Firebase Storage. Allows tracking quotation value, currency, probability of close, and related details.
- Activity Tracking and Reminders: Track all activities related to a client, including contact history, follow-up dates, and internal notes. Automated reminders for upcoming follow-ups using a tool.
- Sales Performance Dashboard: Interactive dashboard providing key performance indicators (KPIs) such as quotations sent, active opportunities, sales closed, conversion rates, and sales by region/seller.
- Access Control: Implement security model where sellers only have access to their data. Admins and managers have access to all data.

## Style Guidelines:

- Primary color: Deep red (#8B0000) to convey power and importance, aligned with the logo.
- Secondary color: Black (#000000) to provide a strong contrast and a sense of sophistication.
- Accent color: Light gray (#D3D3D3) used sparingly for calls to action and key interactive elements to provide a subtle highlight without clashing with the primary colors.
- Body and headline font: 'Roboto', a sans-serif font known for its readability and clean design, making it suitable for both headings and body text.
- Use a set of consistent, professional icons in black and white to complement the color scheme. Icons should be minimalist and represent key actions and data points.
- Maintain a clean, structured layout with clear sections for different data types. Prioritize key metrics and actions on dashboards. Use a card-based design for better organization.
- Incorporate subtle animations to enhance user experience, such as transitions when moving deals across the Kanban board. Animations should be fast and purposeful.
